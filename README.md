# Système de Gestion de Bibliothèque 📚

Une application de bureau (Desktop) développée en **Python** permettant de gérer un stock de livres, les abonnés et les emprunts via une interface graphique moderne et une base de données MySQL.

## ✨ Fonctionnalités
* **Gestion des Livres** : Ajouter, modifier, supprimer et rechercher des ouvrages.
* **Interface Intuitive** : Navigation par onglets (Livres, Abonnés, Emprunts, Admin).
* **Base de Données** : Connexion en temps réel avec MySQL via PyMySQL.
* **Affichage Dynamique** : Visualisation des données dans un tableau (Treeview).

## 🛠️ Technologies utilisées
* **Langage** : Python 3.x
* **GUI** : Tkinter
* **SGBD** : MySQL
* **Connecteur** : PyMySQL
